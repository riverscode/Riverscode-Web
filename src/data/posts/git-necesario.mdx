## Trabajar con ramas

### Crear una rama

```bash:terminal
git checkout -b nombre-rama
```

### Cambiar de rama

```bash:terminal
git checkout nombre-rama
```

### Eliminar una rama

```bash:terminal
git branch -d nombre-rama
```

### Eliminar una rama forzada


```bash:terminal
git branch -D nombre-rama
```

### Ver las ramas

```bash:terminal
git branch
```

### Ver las ramas y sus commits

```bash:terminal
git branch -v
```
