## ¿Qué es el testing?

El testing es una técnica de desarrollo de software que permite verificar que el código que se ha escrito cumple con las especificaciones que se han definido. Es decir, que el código que se ha escrito hace lo que se espera que haga.

## ¿Por qué es importante?

El testing es una técnica muy importante en el desarrollo de software, ya que permite verificar que el código que se ha escrito hace lo que se espera que haga. Esto es muy importante, ya que si el código no hace lo que se espera que haga, el software no funcionará correctamente.

## ¿Cómo se hace?

El testing se hace de dos formas:

1. **Testing manual:** El desarrollador escribe el código y lo prueba manualmente para verificar que hace lo que se espera que haga.

2. **Testing automatizado:** El desarrollador escribe el código y escribe pruebas para verificar que hace lo que se espera que haga.

## ¿Qué es una prueba?

Una prueba es un conjunto de instrucciones que se ejecutan para verificar que el código hace lo que se espera que haga. Por ejemplo, si se está desarrollando una función que suma dos números, una prueba podría ser:

```js:suma.js
const sum = (a, b) => a + b;

const result = sum(1, 2);

if (result !== 3) {
  throw new Error("La suma no es correcta");
}
```

En este caso, la prueba verifica que la función `sum` hace lo que se espera que haga, es decir, que suma dos números.

## ¿Qué es una aserción?

Una aserción es una instrucción que verifica que una condición se cumpla. Por ejemplo, en el ejemplo anterior, la aserción es la instrucción `if (result !== 3) { ... }`.

## ¿Qué es un test runner?

Un test runner es un programa que ejecuta las pruebas que se le pasan. Por ejemplo, en el ejemplo anterior, el test runner es el programa que ejecuta la prueba.

## ¿Qué es un framework de testing?

Un framework de testing es un conjunto de herramientas que facilitan el desarrollo de pruebas. Por ejemplo, un framework de testing puede proveer una sintaxis para escribir pruebas, un test runner para ejecutar las pruebas, y una forma de organizar las pruebas.

## ¿Qué es Jest?

Jest es un framework de testing para JavaScript. Jest provee una sintaxis para escribir pruebas, un test runner para ejecutar las pruebas, y una forma de organizar las pruebas.
